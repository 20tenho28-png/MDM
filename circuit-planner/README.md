# Circuit Planner — EU Edition

Open `circuit_planner.html` in any browser. That's the whole app.

- `npm test` — full headless test suite (no dependencies needed)
- `npm run sheets` — render visual verification sheets to `tools/out_*`
  (SVG always; `npm i sharp` once for PNGs)
- `CLAUDE.md` — architecture, rules and workflow for Claude Code

Working in Claude Code: open this folder and just describe the next batch.
Optional: `git init && git add -A && git commit -m "v6 baseline"` for history.
